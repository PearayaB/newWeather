# Weather App

# About the project
A weather app that can display temperatures and PM 2.5 values in one app from a user's input some city name.

# Feature
* Register & Login
* Search cityname/country
* Favorite
* Pictured changes by city name

# Developed with
* ejs
* node 
* phpMyAdmin
